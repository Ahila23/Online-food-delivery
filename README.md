# Online-food-delivery
Analyzed online food delivery data using Python, SQL, and Power BI to uncover insights on customer ordering patterns, revenue trends, delivery performance, and cancellations. Built dashboards and KPIs to help businesses improve operations, optimize delivery time, and enhance customer satisfaction.
